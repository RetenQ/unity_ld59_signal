> I wanted to submit a version early so I wouldn’t end up rushing everything at the last minute later.

This is a 2D platforming puzzle game. In it, the player must follow (or figure out how *not* to follow) the signal from below in order to reach the goal.

Controls:

* A / D to move
* Space or W to jump
* Ctrl to dash
* Holding jump allows you to jump higher

I’ve always worked in collaboration before, mainly on programming, so this is my first time participating in a Compo as a solo developer. I aimed to create a puzzle game centered around gameplay and mechanics, with simple controls and systems. However, due to time constraints, I don’t feel I was able to design particularly strong puzzles this time.

In any case, I hope you enjoy playing it!

You can download it here: [https://retenq.itch.io/ld59-obeythesignal](https://retenq.itch.io/ld59-obeythesignal)


##### My Story

Hey, if you’re interested, you can also hear a bit about my story.

It’s actually been a long time since I last joined a game jam—my previous one was LD49. Back then, I had just started my undergraduate studies, and quite a lot has happened since.

Game jams marked both the beginning and, in a sense, the end of my game development journey during my undergraduate years. After LD49, I didn’t participate in any more jams for various reasons. I still joined competitions with others and was lucky enough to win some placements in game company-hosted events. But over time, I started to feel that game development wasn’t as fun as it used to be (maybe you’ve heard of the “overjustification effect”?). Still, I wouldn’t say I quit—I just gradually changed how I saw things.

At first, I planned to go straight into the game industry after graduating. Even though I never completely stopped making games, the more competitions I joined, the more I felt that “turning game development into a career was slowly draining my love for it.” So I ran away from that idea. I wanted to keep game dev as a hobby, something I do in my spare time—that felt like the only way to preserve my passion.

So later on (which is now), I went on to do a master’s degree, thinking I could still make games in my free time. But the reality is that I’ve been too busy (still am, lol), or more honestly, I kept convincing myself that “I’m too busy.” The result is that I haven’t really developed any games in the past two years since graduating from undergrad.

My undergraduate final project was a game development project—that was actually the last game I made before LD59.

Recently, I told myself I couldn’t keep going like this. So I decided to start again. I hope joining LDGJ this time can rekindle my passion for game development—or at least warm me up again. Even though I think this project is just okay, I feel like I’ve started making games again. I feel that spark again—and that’s what game jams are for, right?

But no one can deny time. Some of the friends I used to make games with are still in the industry. Some are working on AI-related projects in game companies. Some, like me, went on to pursue completely different graduate studies. And some have left game development entirely. From LD49 to LD59, the world has also changed—we have to acknowledge how AIGC is reshaping everything around us.

Anyway, I don’t want to overthink things anymore. I make games—starting again, and continuing on.
